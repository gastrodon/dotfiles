# X11 Configuration Nix Module

## Overview
This module manages X11 window system configuration, including initialization scripts and X resources for terminal emulators and other X applications.

## Files Encompassed
- `.xinitrc` - X initialization script that starts i3 window manager
- `.Xresources` - X resources database for xterm, URxvt, rofi, and Xcursor configuration

## Module Location
`module/x11.nix`

## Key Configuration Areas

### X Initialization (.xinitrc)
- System and user X resources merging
- System and user key mappings (xmodmap)
- Execution of system xinit scripts
- i3 window manager as the default session

### X Resources (.Xresources)

#### Color Scheme
- Gruvbox color scheme for terminals
- Background: #282828
- Foreground: #ebdbb2
- Full 16-color palette defined

#### Rofi Configuration
- Font: iosevka ss04 12
- Custom color scheme matching terminal colors
- Padding and styling options
- Keyboard shortcuts (Escape, Alt+F1 for cancel)

#### XTerm Configuration
- Terminal name: xterm-256color
- Font: iosevka term ss04, size 11
- Geometry: 90x34
- Save lines: 2000
- Login shell enabled
- Eight-bit input disabled

#### URxvt Configuration
- Save lines: 15000
- Terminal name: rxvt-256color
- Semi-transparent background ([95]#1C2023)
- Font: iosevka term ss04, size 13
- No scrollbar
- Internal border: 15px
- Letter spacing: -1
- Perl extensions: clipboard, url-select, keyboard-select
- Clipboard integration with xclip
- Custom keybindings (m-c for copy, m-v for paste)
- URL launcher: exo-open
- Tab bar colors configured

#### Cursor
- Xcursor.size: 18

## Home Manager Integration Points
- xsession.enable for X session management
- xsession.initExtra for xinitrc commands
- xsession.windowManager.i3 for i3 configuration reference
- xresources.properties for X resources
- xresources.extraConfig for complex resource definitions

## Dependencies
- xorg.xrdb - X resources database manager
- xorg.xmodmap - Keyboard mapping utility
- i3 - Window manager
- iosevka font family (term ss04 variant)
- rofi - Application launcher
- xterm - Terminal emulator
- rxvt-unicode (URxvt) - Alternative terminal emulator
- xclip - Clipboard utility for URxvt
- exo-open or similar for URL launching
- xcursor themes

## Notes
- The color scheme is Gruvbox, which should be considered when theming
- Font "iosevka term ss04" is heavily used and must be available
- Both xterm and URxvt are configured; consider which one is primary
- Rofi configuration may conflict with standalone rofi module if created
- The xinitrc script merges system resources, ensure this doesn't conflict with Home Manager's xresources management
- Consider whether to use a display manager instead of startx from .zprofile
- URxvt transparency requires a compositor
